# Greenwash Check

AI-powered greenwashing assessment tool for Canadian businesses.

## Features
- Assess environmental claims against Bill C-59 and Competition Bureau guidelines
- Upload PDF documents for comprehensive analysis
- Text-based claim assessment
- Customizable assessment criteria

## Environment Variables
- `OPENAI_API_KEY` - Your OpenAI API key
- `PORT` - Server port (default: 5192)

## Deployment
This app is designed to be deployed on Railway or similar Node.js hosting platforms.

```bash
npm install
npm run build
node server.js
```
