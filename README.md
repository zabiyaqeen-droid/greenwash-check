# Greenwash Check

AI-powered greenwashing assessment tool for Canadian businesses to ensure compliance with Bill C-59 and Competition Bureau guidelines.

## Features
- Assess environmental claims against Bill C-59 and Competition Bureau's 6 Principles for Environmental Claims
- Upload PDF documents (up to 200 pages) for comprehensive analysis
- AI-powered analysis with high-detail vision for charts, graphs, and visual content
- Detailed scoring with specific document examples and page references
- Text-based claim assessment
- Customizable assessment criteria

## Environment Variables
- `OPENAI_API_KEY` - Your OpenAI API key
- `PORT` - Server port (default: 5192)

## Deployment

This app is deployed on Render.

### Local Development

```shell
npm install
npm run dev
```

### Production Build

```shell
npm install
npm run build
node server.js
```
